Developing Optimal Control Technology for Distributed Energy Resources (DOCTdER)

CAPS, CISCOR & Nhu Energy, Inc.

  • Development of a novel optimal control solution designed to solve the local and grid-connected distributed energy resources (DER) integration problem.
  • The developed framework is capable of controlling multiple DERs based on forecasted values and real-time energy prices.
  • Software developed using C/C++ and Python.
  • Tested and validated using controller hardware-in-the-loop (CHIL) real-time simulation.

Home Energy Management System (HEMS)

Senior Design Project

The main goal of the Home Energy Management System is to give the user total management and control of his own energy consumption, and also reduce the demand load with the use of Real-Time Pricing (RTP). By using Real-Time Pricing (RTP) provided by electric utility companies, consumers have the ability to adjust their energy usage in order to reduce their electricity bill.

Project Features:

  • Automates the power consumption of a client by controlling loads according to a real-time price.
  • Web application implemented using PHP, Javascript, and HTML. Load controllers implemented using C/C++
  • Main and load controllers implemented in C/C++.
  • Communication via 2.4 Ghz RF/Wireless technology.
  • Real-Time Web interface control
  • This project was awarded the "Best Environmental" Prize on the TI Innovation Challenge 2016
