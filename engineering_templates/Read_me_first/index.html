Starting out

Think of the Internet as a bunch of connected computers. Here at the College of Engineering, we place many of our Internet files on a computer named codex2.The name of the server is codex2. A Server is a computer that has many different uses and functions. In this case, it’s used to hold files and serve them when called upon. Every time you type "www.eng.fsu.edu" into your browser (that’s Internet Explorer, Firefox, Chrome, etc), your computer sends a message to our computers. It says something along the lines of, “Hey, I need this file and it should be here at this address.” Then our computer sends it over. All of this happens really fast. Those Internet files are written in html.

The purpose of a web browser is to read HTML documents and compose them into visible web pages. HTML is written in the form of elements. It consists of tags enclosed in angle brackets like <html>. HTML tags most commonly come in pairs like <h1> and </h1>, although some tags are unpaired, for example <img>. The first tag in a pair is the start tag, and the second tag is the end tag.
