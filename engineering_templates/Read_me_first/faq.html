FAQ

Why can’t I see my network drive?
If its not there, right click Computer, click map network drive, Select Z drive or any other drive that might already have you credentials pre-stored. Then in the folder box, type \\codex2.eng.fsu.edu\YourUserName. Click finish and okay to any pop-up messages.

How do I connect from home?
You will need to install an SFTP client on your computer. We recommend FileZilla. Go to Filezilla and click the "Download FileZilla Client" link and select the download appropriate for your operating system. Install it with the default options and start FileZilla.

  • In the "Host" field type "wolf.eng.fsu.edu."
  • In the "User Name" field, type your engineering account username. In the "Password" field, type your password.
  • In the "Port" field, type "22."
You should now be able to drag and drop files between the left and right panes. Left reflects your local computer and right reflects your College of Engineering files.

Why aren’t my pictures showing up?
Pictures have file extensions like .jpg, .jpeg, .gif, .png, and .bmp. Make sure the name is something user friendly like “pic1.jpg”. A picture like “My First Vacation (1) (2).jpg” will trip you up because a browser reads a space as “%20”. A picture saved from Facebook, for example, “1013382_605109589522303_1954667314_n.JPEG” is super long. A computer will follow your instructions down to the very last dot. So remember capitals, spaces, and symbols count. In this html tag: <img src=“pic1.jpg”>, make sure you've placed the correct file name. If you put your photos in a folder, make sure the address is correct. For example, <img src=“vacation/pic1.jpg”>.
