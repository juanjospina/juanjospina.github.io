Connecting

We're going to connect to our servers. Let’s access your files on codex2. Sign into a computer using your engineering user name and password. If you’re working from home, follow these instructions . Find My Computer in the start bar or in a windows file explorer window. Under network location you should see your user name connected to codex2. Click that, then go into public_html.

All of the files in this folder are viewable live in an internet browser at this address: http://eng.fsu.edu/~YourUserName.

Servers and browsers default to index.html so this will be the first page you see when you type in a web address. Lets say there is a folder inside of your public_html called cats and there is an index.html file in there, this address will default to the index: http://eng.fsu.edu/~YourUserName/cats. When naming folders and files, remember to keep it simple. Don’t use spaces or symbols. You don’t have to type public_html, we see that because we’re connected to the server.
