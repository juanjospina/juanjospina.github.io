Editing

What will you need?

  • A text editor like Notepad (Windows), TextEdit (Mac), or Notepad ++ (all systems).
  • Access to your network drive (This is your engineering account. We went over this here).
  • Special note for Mac users: To use text edit as an html editor change the configurations as explained here.

In your network drive, you’ll find template files inside a folder called public_html. Save the folder and its contents onto your desktop or anywhere else that is easily accessible. Open these files with your text editor. You can right click the files and choose to open them with your text editor. Start with index.html since your browser defaults to this page. You will see a basic html page.

The basic structure is:

<!DOCTYPE html> This tells the browser we’re sending an html file.

<html> We're opening the html file tag.

<head> Head elements contain things like window titles and links to style sheets. Style sheets give the text on your page styling. It uses a language called CSS. You don’t have to worry about this because we’ve provided styling in a CSS styling sheet. This file is called bootstrap.css. There's another one called bootstrap-responsive.css. This is the style used for tablets and smart phones.

</head> Now we’re closing the head tag.

<body> The body element contains all items viewable by the browser. This is where all of your content goes. We’ve provided you with typical body elements. All you need to do is edit the text within the elements.

</body> Now we’re closing the body tag.

</html> We're closing the html tag. The html file is over.

There will be comments to guide you in your editing efforts. Be careful for typos! Save a clean copy of your template just in case you want to go back and find any errors. Or you can save different versions of your folders. For example: site1, site2, etc. When you are done editing, make sure to save the final version inside of the public_html folder. This makes it viewable.

Html Comments: <!--This is a comment. Comments are not displayed in the browser. I'll place notes in here to guide you.-- >

A special note for passport.html:

The images should be sized 400 x 400 pixels. Make sure you save any pictures you edit to your computer and move them onto your network drive. Save your pictures with simple names. You can place them inside of public_html. Or, for example, you can save a picture within a folder called pictures, within public_html. The address would be: "http://eng.fsu.edu/~YourUserName/pictures/picName.jpg." You can resize your pictures using these sites:
  • Picresize
  • Fotor
  • Pixlr
  • Be Funky
